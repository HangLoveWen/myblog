1.语法

SELECT语句用于从表中查询数据。执行的结果被存储在一个结果表中(称为结果集)。语法格式如下：

SELECT * FROM 表名称

SELECT 列名称 FROM 表名称

注意：SQL语句中的关键字对大小写不敏感。SELECT等效于SELECT，FROM等效于from。
